# 🎓 ҚАРСУ Student Bot — Инструкция по запуску

> Эта инструкция написана для тех, кто не программист. Просто следуйте шагам!

---

## 📋 ЧТО У ВАС ПОЛУЧИТСЯ
- Telegram бот с кнопкой "Личный кабинет"
- Красивое приложение внутри Telegram
- Страницы: Оценки, Расписание, Посещаемость, Профиль

---

## 🚀 ШАГ 1 — Создать бота в Telegram (5 минут)

1. Откройте Telegram, найдите **@BotFather**
2. Отправьте: `/newbot`
3. Введите **название** бота (например: `ҚАРСУ Студент`)
4. Введите **username** бота (например: `karsu_student_uz_bot`)
5. BotFather даст вам **токен** — он выглядит так:
   ```
   1234567890:AAFxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   ```
6. **СОХРАНИТЕ ЭТОТ ТОКЕН** — он понадобится в Шаге 3!

---

## 🌐 ШАГ 2 — Зарегистрироваться на Railway (бесплатный хостинг)

1. Зайдите на сайт: **https://railway.app**
2. Нажмите **"Login"** → **"Login with GitHub"**
3. Если нет GitHub — зарегистрируйтесь на **https://github.com** (бесплатно)
4. После входа вы окажетесь на главной странице Railway

---

## 📁 ШАГ 3 — Загрузить проект на GitHub

1. Зайдите на **https://github.com**
2. Нажмите **"New repository"** (зелёная кнопка)
3. Название: `karsu-bot` → нажмите **"Create repository"**
4. Загрузите ВСЕ файлы этой папки в репозиторий:
   - `main.py`
   - `bot.py`
   - `requirements.txt`
   - `Dockerfile`
   - `railway.json`
   - папку `frontend/` с файлом `index.html`

> 💡 Чтобы загрузить файлы: нажмите "uploading an existing file" на странице репозитория

---

## ⚙️ ШАГ 4 — Деплой на Railway

1. На сайте Railway нажмите **"New Project"**
2. Выберите **"Deploy from GitHub repo"**
3. Выберите ваш репозиторий `karsu-bot`
4. Railway начнёт деплой — подождите 2-3 минуты
5. После успешного деплоя нажмите на проект
6. Зайдите в **Settings → Networking → Generate Domain**
7. Скопируйте URL — он будет выглядеть так:
   ```
   https://karsu-bot-production.railway.app
   ```

---

## 🔑 ШАГ 5 — Добавить секретные данные (переменные)

1. В Railway зайдите в ваш проект
2. Нажмите **"Variables"**
3. Добавьте две переменные:

| Имя переменной | Значение |
|---|---|
| `BOT_TOKEN` | Токен из Шага 1 |
| `WEBAPP_URL` | https://ваш-url.railway.app/app |

4. Нажмите **"Deploy"** чтобы перезапустить с новыми данными

---

## 🤖 ШАГ 6 — Настроить кнопку в BotFather

1. Откройте **@BotFather** в Telegram
2. Отправьте `/mybots`
3. Выберите вашего бота
4. Нажмите **"Bot Settings"** → **"Menu Button"**
5. Введите URL вашего сайта: `https://ваш-url.railway.app/app`

---

## ✅ ГОТОВО!

Откройте вашего бота в Telegram, нажмите **START** и затем кнопку **"Личный кабинет"**!

---

## ❓ ЧАСТЫЕ ВОПРОСЫ

**Q: Не работает авторизация**
A: Сайт student.karsu.uz может изменить структуру. Свяжитесь с разработчиком.

**Q: Данные не загружаются**
A: Убедитесь что BOT_TOKEN и WEBAPP_URL правильно указаны в Railway Variables.

**Q: Бот не отвечает**
A: Проверьте логи в Railway → нажмите на сервис → вкладка "Logs".
